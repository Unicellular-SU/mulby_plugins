## 🔧 详细使用指南

### 1. 使用常用模式
1. **展开面板**：点击"常用模式"标题旁的展开按钮
2. **选择模式**：点击任意模式卡片
3. **自动应用**：正则表达式和标志会自动填入
4. **开始测试**：在测试文本区输入文本查看匹配结果

### 2. 切换匹配标志
1. **查看标志**：在侧边栏标志面板查看所有可用标志
2. **了解功能**：悬停在标志上查看详细说明
3. **切换状态**：点击标志切换激活状态
4. **组合使用**：可以同时激活多个标志

### 3. 编辑正则表达式
1. **直接输入**：在正则输入框中输入表达式
2. **使用分隔符**：表达式会自动显示在/ /之间
3. **设置标志**：在标志输入框输入标志字符
4. **实时验证**：输入时自动验证语法

### 4. 测试文本匹配
1. **输入文本**：在测试文本区输入或粘贴文本
2. **实时匹配**：输入时自动执行匹配
3. **查看结果**：匹配内容会自动高亮显示
4. **分析详情**：查看每个匹配的详细信息

### 5. 使用操作功能
1. **复制正则**：点击"复制正则"按钮复制完整表达式
2. **复制匹配**：点击"复制匹配"按钮复制所有匹配结果
3. **查看反馈**：操作成功会有通知提示

## 💡 高级技巧

### 1. 正则表达式优化
- **使用分组**：用()创建捕获组
- **应用量词**：使用* + ? {n,m}控制匹配次数
- **使用边界**：\b匹配单词边界
- **字符类**：[a-z]匹配字符范围

### 2. 性能优化
- **避免贪婪匹配**：使用*? +?进行非贪婪匹配
- **使用具体字符类**：避免过度使用.
- **合理使用锚点**：^ $匹配行首行尾
- **预查断言**：使用(?=) (?!)进行预查

### 3. 特殊字符处理
- **转义特殊字符**：使用\转义. * + ?等
- **Unicode支持**：使用u标志支持Unicode
- **多行文本**：使用m标志处理多行文本
- **点号匹配换行**：使用s标志让.匹配换行符

## 🔧 故障排除

### 常见问题
#### Q1: 正则表达式不匹配怎么办？
- 检查正则表达式语法是否正确
- 确认标志设置是否合适
- 查看错误提示信息
- 使用常用模式作为参考

#### Q2: 如何匹配多行文本？
- 使用m标志启用多行模式
- 使用^和$匹配行首行尾
- 使用s标志让.匹配换行符

#### Q3: 插件没有响应怎么办？
- 检查插件是否已正确安装
- 重启Mulby应用程序
- 查看控制台错误信息

#### Q4: 如何提高匹配性能？
- 避免使用.*的过度匹配
- 使用更具体的字符类
- 合理使用锚点和边界

### 错误处理

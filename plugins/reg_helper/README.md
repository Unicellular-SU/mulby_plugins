# reg_helper - iOS 26 拟态玻璃效果正则表达式编辑器

一个基于苹果iOS 26设计语言的现代化正则表达式编辑器插件，采用拟态玻璃效果（毛玻璃效果）设计，提供优雅、直观的用户体验。

## ✨ 设计特色

### 🎨 iOS 26 拟态玻璃效果
- **毛玻璃背景**：采用半透明背景和模糊效果
- **柔和阴影**：精致的阴影层次感
- **流畅动画**：平滑的过渡和交互反馈
- **现代配色**：苹果风格的色彩系统
- **响应式布局**：适配不同屏幕尺寸

### 🎯 核心功能
- **实时正则表达式编辑**：输入时自动匹配
- **常用模式库**：10个预设正则表达式模式
- **标志切换**：支持g/i/m/s/u/y所有正则标志
- **结果高亮**：匹配内容实时高亮显示
- **详细分析**：显示匹配位置、长度和捕获组
- **错误验证**：实时语法检查和友好提示

## 🚀 快速开始

### 安装插件
1. 将插件文件夹复制到Mulby的插件目录
2. 重启Mulby或刷新插件列表
3. 插件将自动出现在可用插件中

### 启动方式
- **方式1**：在Mulby主界面输入 `regex`
- **方式2**：在Mulby主界面输入 `正则`
- **方式3**：通过快捷键（如果配置了快捷键）

## 🔧 功能详解

### 1. 常用模式面板
- **一键应用**：点击模式卡片快速应用正则表达式
- **图标标识**：每个模式都有对应的图标
- **预览显示**：显示正则表达式预览
- **收起/展开**：可折叠的面板设计

### 2. 标志面板
- **视觉反馈**：激活的标志有特殊样式
- **工具提示**：悬停显示标志说明
- **快速切换**：点击切换标志状态

### 3. 正则输入面板
- **语法高亮**：实时验证和状态显示
- **分隔符**：使用/作为正则表达式分隔符
- **标志输入**：独立的标志输入框

### 4. 测试文本面板
- **多行输入**：支持大段文本输入
- **实时匹配**：输入时自动执行匹配
- **语法检查**：实时验证正则表达式

### 5. 匹配结果面板
- **高亮显示**：匹配内容在文本中高亮
- **匹配计数**：显示匹配数量
- **详细分析**：显示每个匹配的详细信息

### 6. 操作面板
- **复制正则**：复制完整的正则表达式
- **复制匹配**：复制所有匹配结果
- **视觉反馈**：操作按钮有悬停和点击效果

### 7. 信息面板
- **完整正则**：显示当前正则表达式
- **转义字符串**：显示转义后的模式
- **状态指示**：显示正则表达式是否有效

## 🎮 使用示例

### 示例1：提取邮箱地址
1. 点击侧边栏"邮箱地址"模式卡片
2. 在测试文本区粘贴包含邮箱的文本
3. 查看高亮显示的邮箱地址
4. 点击"复制匹配"按钮复制所有邮箱

### 示例2：验证手机号码
1. 点击侧边栏"手机号码"模式卡片
2. 输入要验证的手机号码
3. 查看是否匹配成功
4. 根据需要调整正则表达式

### 示例3：清理HTML标签
1. 点击侧边栏"HTML标签"模式卡片
2. 粘贴HTML文本
3. 查看匹配的HTML标签
4. 使用替换功能清理标签

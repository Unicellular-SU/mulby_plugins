# 图片拼接插件测试指南

## 插件功能概述
已成功创建名为 "image_stitching" 的图片拼接插件，该插件可以将多张图片任意纵向裁切后拼接成一张长图。

## 核心功能

### 1. 图片管理
- 支持上传多张图片（PNG、JPG、JPEG、GIF、WebP、BMP格式）
- 支持拖放上传
- 支持通过插件命令直接传入图片

### 2. 图片裁切
- 每张图片可独立设置顶部和底部裁切比例（0-100%）
- 实时预览裁切效果
- 显示裁切后的实际像素高度

### 3. 图片排序
- 可调整图片的拼接顺序（上移/下移）
- 可删除不需要的图片

### 4. 拼接设置
- **输出宽度**：可设置生成图片的宽度（100-5000px）
- **图片间距**：可设置图片之间的间距（0-100px）
- **背景颜色**：可设置拼接图片的背景色

### 5. 结果导出
- **复制到剪贴板**：将生成的图片复制到系统剪贴板
- **下载图片**：将生成的图片保存为PNG文件

## 触发方式

### 命令触发
1. 输入 `图片拼接` 打开插件界面
2. 输入 `image stitch` 打开插件界面

### 文件触发
1. 直接拖放图片文件到插件
2. 选择图片文件后调用插件

## 使用流程

1. **添加图片**：
   - 点击"添加图片"按钮或拖放图片文件
   - 支持多选图片文件

2. **调整裁切**：
   - 使用滑块调整每张图片的顶部和底部裁切
   - 实时预览裁切效果

3. **设置参数**：
   - 设置输出图片宽度
   - 调整图片间距
   - 选择背景颜色

4. **生成图片**：
   - 点击"开始拼接"按钮
   - 等待处理完成

5. **导出结果**：
   - 点击"复制到剪贴板"复制图片
   - 点击"下载图片"保存文件

## 技术特点

### 前端技术
- React 18 + TypeScript
- Canvas API 用于图片处理
- 响应式设计，支持移动端
- 主题适配（亮色/暗色）

### 图片处理
- 使用HTML5 Canvas进行图片裁切和拼接
- 保持图片原始比例
- 支持透明背景

### 用户体验
- 实时预览裁切效果
- 进度提示和错误处理
- 直观的操作界面

## 注意事项

1. **图片格式**：支持常见图片格式，但建议使用PNG或JPEG以获得最佳效果
2. **图片大小**：建议单张图片不超过10MB
3. **输出尺寸**：生成的图片尺寸受设备内存限制
4. **浏览器兼容**：需要支持Canvas API的现代浏览器

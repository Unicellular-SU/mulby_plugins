# 剪贴板历史插件

查看和管理剪贴板历史记录，支持文本、图片、文件。

## 功能特性

- ✅ **自动记录** - 自动记录所有剪贴板变化（文本、图片、文件）
- ✅ **按时间倒序** - 最新的记录显示在最前面
- ✅ **类型过滤** - 按文本、图片、文件类型过滤
- ✅ **搜索功能** - 快速搜索文本内容和文件名
- ✅ **收藏功能** - 收藏重要的记录，永久保留
- ✅ **一键复制** - 点击即可复制历史记录到剪贴板
- ✅ **统计信息** - 显示各类型记录的数量
- ✅ **暗色主题** - 自动跟随系统主题

## 触发方式

- `剪贴板` - 打开剪贴板历史
- `clipboard` - 打开剪贴板历史
- `cb` - 打开剪贴板历史（快捷方式）

## 使用说明

### 查看历史

1. 输入触发词（如 `剪贴板`）打开插件
2. 历史记录按时间倒序显示
3. 支持查看文本、图片预览、文件列表

### 过滤和搜索

- **类型过滤**：点击顶部的"全部"、"文本"、"图片"、"文件"、"收藏"按钮
- **文本搜索**：在搜索框中输入关键词，支持搜索文本内容和文件名

### 操作记录

每条记录提供三个操作按钮：

- **收藏** ⭐ - 收藏/取消收藏记录（收藏的记录不会被自动清理）
- **复制** 📋 - 复制到剪贴板并自动关闭窗口
- **删除** 🗑️ - 删除单条记录

### 清空历史

点击右上角的"清空历史"按钮可以清空所有历史记录（收藏的记录会保留）。

## 数据存储

- 历史记录存储在 SQLite 数据库中
- 默认保留最近 1000 条记录
- 收藏的记录永久保留
- 图片最大 5MB

## 技术实现

- **前端**：React + TypeScript + Tailwind CSS
- **后端**：Node.js + SQLite
- **剪贴板监听**：Native Addon（零性能开销）
- **主题**：自动跟随系统主题

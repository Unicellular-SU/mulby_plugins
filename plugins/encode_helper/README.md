# 编码小助手 (encode_helper)

一个功能全面的编码转换工具插件，集成了8大常用编码转换功能。

## ✨ 核心功能

### 1. ⏰ 时间转换
- 时间戳 ↔ 日期时间相互转换
- 支持秒级和毫秒级时间戳
- 自动识别输入格式

### 2. 🔢 Base64编解码
- **加密（编码）**：将文本转换为Base64格式
- **解密（解码）**：将Base64还原为原文本
- 支持中文字符处理
- 独立的加密和解密按钮

### 3. 🔗 URL编解码
- **编码**：将文本转换为URL编码格式
- **解码**：将URL编码还原为原文本
- 处理特殊字符和中文
- 独立的编码和解码按钮

### 4. 🔐 哈希加密
- 支持多种哈希算法：MD5、SHA1、SHA256、SHA512
- 使用Web Crypto API实现
- 实时计算哈希值

### 5. 🔤 Unicode转换
- **编码**：将文本转换为Unicode格式（\uXXXX）
- **解码**：将Unicode还原为原文本
- 独立的编码和解码按钮

### 6. 🆔 UUID生成
- 生成标准UUID v4
- 一键生成并复制
- 符合RFC 4122标准

### 7. 🔢 进制转换
- 二进制、八进制、十进制、十六进制相互转换
- 灵活选择源进制和目标进制
- 自动识别输入格式

### 8. 📄 HTML转义
- **编码**：将HTML特殊字符转换为实体
- **解码**：将HTML实体还原为原字符
- 独立的编码和解码按钮

## 🚀 使用方法

### 安装步骤
1. 将插件文件夹复制到Mulby的插件目录
2. 重启Mulby或重新加载插件
3. 在插件管理中启用"编码小助手"

### 触发方式
在Mulby主界面输入以下任意关键词：
- `时间转换`、`timestamp`
- `base64`、`base64编码`、`base64解码`
- `url编码`、`url解码`
- `哈希`、`md5`、`sha1`、`sha256`
- `unicode`、`unicode编码`、`unicode解码`
- `uuid`、`生成uuid`
- `进制转换`、`二进制`、`十六进制`
- `html转义`、`html编码`、`html解码`

### 界面操作
1. 通过顶部标签页切换功能
2. 在输入框输入内容
3. 点击对应的操作按钮：
   - **Base64**：点击"加密"或"解密"按钮
   - **URL**：点击"编码"或"解码"按钮
   - **Unicode**：点击"编码"或"解码"按钮
   - **HTML**：点击"编码"或"解码"按钮
   - 其他功能：点击"转换"按钮
4. 结果自动复制到剪贴板

## 🔧 开发说明

### 技术栈
- **前端**: React 18 + TypeScript
- **构建工具**: Vite + esbuild
- **样式**: CSS变量 + 响应式设计
- **API**: Mulby插件API

### 构建命令
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建插件
npm run build

# 打包插件
npm run pack
```

### 扩展功能
如需添加新功能：
1. 在`src/main.ts`中注册新功能
2. 在`src/ui/App.tsx`中添加处理逻辑
3. 更新UI界面和样式
4. 更新配置文件

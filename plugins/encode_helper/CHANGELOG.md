# 更新日志

## v1.1.0 (2024-01-20)

### 🎉 重要改进：界面优化

#### Base64功能改进
- **新增**：独立的"加密"和"解密"按钮
- **加密按钮**：将用户输入转换为Base64格式
- **解密按钮**：将Base64还原为原文本
- **移除**：自动检测功能，避免混淆

#### URL功能改进
- **新增**：独立的"编码"和"解码"按钮
- **编码按钮**：将文本转换为URL编码格式
- **解码按钮**：将URL编码还原为原文本

#### Unicode功能改进
- **新增**：独立的"编码"和"解码"按钮
- **编码按钮**：将文本转换为Unicode格式（\uXXXX）
- **解码按钮**：将Unicode还原为原文本

#### HTML功能改进
- **新增**：独立的"编码"和"解码"按钮
- **编码按钮**：将HTML特殊字符转换为实体
- **解码按钮**：将HTML实体还原为原字符

### 🚀 用户体验提升
- **更直观的操作**：用户明确知道点击哪个按钮执行什么操作
- **减少错误**：避免自动检测可能导致的错误转换
- **界面一致性**：所有编码/解码功能都采用相同的按钮设计
- **更好的反馈**：每个按钮都有明确的功能标签

### 🔧 技术改进
- **代码重构**：将通用处理函数拆分为特定功能函数
- **状态管理优化**：改进状态更新逻辑
- **错误处理增强**：提供更详细的错误信息
- **性能优化**：减少不必要的重新渲染

### 📝 文档更新
- **更新README**：详细说明新的界面设计
- **添加测试页面**：创建test_base64.html用于功能验证
- **完善使用说明**：提供更清晰的操作指南

## v1.0.0 (2024-01-20)

### ✨ 初始版本功能
- **时间转换**：时间戳与日期时间相互转换
- **Base64编解码**：支持自动检测编码/解码
- **URL编解码**：支持自动检测编码/解码
- **哈希加密**：支持MD5、SHA1、SHA256、SHA512
- **Unicode转换**：支持自动检测编码/解码
- **UUID生成**：生成标准UUID v4
- **进制转换**：二进制、八进制、十进制、十六进制相互转换
- **HTML转义**：支持自动检测编码/解码

### 🏗️ 技术架构
- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite + esbuild
- **样式系统**：CSS变量 + 响应式设计
- **插件API**：Mulby插件系统集成

### 📦 项目结构
- 完整的TypeScript项目配置
- 模块化的UI组件设计
- 详细的文档和安装指南
- 完整的构建和打包脚本

## 升级说明

### 从v1.0.0升级到v1.1.0
1. **界面变化**：
   - Base64、URL、Unicode、HTML页面现在有独立的操作按钮
   - 不再依赖自动检测功能
   - 用户需要明确选择编码或解码操作

2. **操作方式**：
   - 对于编码/解码功能，现在需要点击对应的按钮
   - 其他功能（时间、哈希、UUID、进制）保持不变

3. **兼容性**：
   - 完全向后兼容
   - 所有现有功能正常工作
   - 插件配置无需更改

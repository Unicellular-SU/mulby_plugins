# 🎨 颜色助手 (color_helper)

一个功能丰富的颜色工具插件，支持颜色解析、图片色卡提取、UI色卡、传统色、渐变色和收藏功能。

## ✨ 功能特性

### 1. **颜色解析**
- 支持 HEX 颜色格式解析（如 `#FF6B6B`）
- 支持 RGB 颜色格式解析（如 `rgb(255, 107, 107)`）
- 实时显示颜色预览
- 一键复制 HEX、RGB、HSL 格式
- 支持添加到收藏

### 2. **图片色卡**
- 上传图片自动提取主色调
- 显示每种颜色的占比百分比
- 一键复制提取的颜色值
- 支持常见图片格式（JPG、PNG、GIF、WebP、BMP）

### 3. **UI色卡**
- 常用 UI 设计颜色库
- 包含主色、辅色、状态色等
- 每种颜色附带使用场景说明
- 一键复制颜色值

### 4. **传统色**
- 中国传统颜色库
- 包含朱红、胭脂、石青等经典颜色
- 显示颜色名称、拼音、朝代信息
- 一键复制颜色值

### 5. **渐变色**
- 预设精美渐变色方案
- 显示渐变 CSS 代码
- 支持收藏喜欢的渐变色
- 一键复制渐变 CSS

### 6. **收藏功能**
- 收藏喜欢的颜色和渐变色
- 统一管理所有收藏内容
- 支持从收藏中移除

## 🚀 使用方法

### 安装插件
1. 将插件文件夹放置在 Mulby 的插件目录中
2. 重启 Mulby 或刷新插件列表
3. 启用 "color_helper" 插件

### 使用方式
1. **通过关键词触发**：
   - 输入 `color` 打开颜色解析工具
   - 输入 `image colors` 打开图片色卡工具
   - 输入 `ui colors` 查看 UI 色卡
   - 输入 `traditional colors` 查看传统色
   - 输入 `gradient` 打开渐变色工具
   - 输入 `color favorites` 查看收藏

2. **直接输入颜色值**：
   - 输入 HEX 颜色（如 `#FF6B6B`）自动解析
   - 输入 RGB 颜色（如 `rgb(255, 107, 107)`）自动解析

3. **拖拽图片文件**：
   - 拖拽图片文件到输入框自动提取颜色

## 🎯 快捷键功能

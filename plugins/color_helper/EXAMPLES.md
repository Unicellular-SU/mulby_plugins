# 颜色助手使用示例

## 🎯 快速开始

### 示例1: 解析HEX颜色
1. 在Mulby输入框中输入：`#FF6B6B`
2. 选择"解析HEX颜色"功能
3. 插件将自动打开并显示：
   - 颜色预览
   - HEX值：`#FF6B6B`
   - RGB值：`rgb(255, 107, 107)`
   - HSL值：`hsl(0, 100%, 71%)`
4. 点击"复制"按钮复制任意格式

### 示例2: 解析RGB颜色
1. 输入：`rgb(74, 144, 226)`
2. 选择"解析RGB颜色"功能
3. 获得对应的HEX值：`#4A90E2`

### 示例3: 提取图片颜色
1. 拖拽一张图片到Mulby输入框
2. 选择"从图片提取颜色"功能
3. 插件将显示：
   - 图片预览
   - 提取的5种主色调
   - 每种颜色的占比
4. 点击颜色卡片复制颜色值

## 📋 完整使用场景

### 场景1: 设计师选择UI颜色
**需求**：为新的设计项目选择主色调

**步骤**：
1. 输入 `ui colors` 打开UI色卡
2. 浏览预设的UI颜色方案
3. 点击"Primary"颜色查看详细信息
4. 点击"复制颜色"获取HEX值
5. 如果需要，点击"❤️ 添加到收藏"

**结果**：快速获得经过验证的UI设计颜色

### 场景2: 提取品牌图片颜色
**需求**：从品牌Logo提取颜色方案

**步骤**：
1. 准备品牌Logo图片（PNG格式）
2. 输入 `image colors` 打开图片色卡工具
3. 上传Logo图片
4. 查看提取的颜色及其占比
5. 将主要颜色添加到收藏
6. 在收藏页面管理所有品牌颜色

**结果**：建立品牌颜色库，方便后续设计使用

### 场景3: 寻找中国传统颜色
**需求**：为传统文化项目寻找合适的颜色

**步骤**：
1. 输入 `traditional colors` 打开传统色库
2. 浏览按朝代和分类组织的颜色
3. 查看"朱红"（明代红色）的详细信息
4. 复制颜色值：`#ED5121`
5. 了解颜色的文化背景和拼音

**结果**：获得具有文化内涵的传统颜色

### 场景4: 创建渐变色背景
**需求**：为网站创建漂亮的渐变色背景

**步骤**：
1. 输入 `gradient` 打开渐变色工具
2. 浏览预设的渐变色方案
3. 选择"日落"渐变查看效果
4. 点击"复制CSS"获取代码：
   ```css
   background: linear-gradient(to right, #FF6B6B, #FFE66D);
   ```
5. 点击"🤍 收藏"保存喜欢的渐变

**结果**：快速获得可直接使用的CSS渐变代码

### 场景5: 管理设计颜色库
**需求**：整理和管理项目中使用的所有颜色

**步骤**：
1. 在不同功能页面收藏需要的颜色
2. 输入 `color favorites` 打开收藏页面
3. 查看所有收藏的颜色和渐变色
4. 使用"复制"按钮快速获取颜色值
5. 使用"移除"按钮清理不再需要的颜色

**结果**：建立个人颜色库，提高设计效率


## 🆘 常见问题

### Q: 支持哪些颜色格式？
A: 目前支持HEX（#RRGGBB）和RGB（rgb(r,g,b)）格式，未来计划支持HSL、CMYK等。

### Q: 图片支持哪些格式？
A: 支持JPG、PNG、GIF、WebP、BMP等常见图片格式。

### Q: 收藏数据保存在哪里？
A: 收藏数据保存在插件的本地存储中，不会上传到服务器。

### Q: 如何导出收藏的颜色？
A: 目前支持通过剪贴板逐个复制，批量导出功能正在开发中。

### Q: 插件是否免费？
A: 是的，颜色助手插件完全免费使用。

## 🔧 核心文件说明

### 1. `src/main.ts` - 插件主入口
- **功能注册**：注册6个主要功能模块
- **生命周期管理**：处理插件的加载、卸载、启用、禁用
- **API集成**：与Mulby主程序通信

### 2. `src/ui/App.tsx` - 主界面组件
- **标签页导航**：6个功能标签页切换
- **颜色解析**：HEX/RGB颜色格式解析
- **图片处理**：图片上传和颜色提取
- **数据管理**：传统色、UI色卡、渐变色数据
- **收藏系统**：颜色和渐变色收藏管理

### 3. `src/ui/styles.css` - 样式文件
- **主题系统**：亮色/暗色主题支持
- **组件样式**：所有UI组件的样式定义
- **响应式设计**：移动端适配
- **动画效果**：过渡动画和交互反馈

### 4. `plugin.json` - 插件配置
- **元数据**：插件名称、版本、描述
- **功能定义**：6个功能模块的配置
- **权限声明**：需要的系统权限
- **图标配置**：插件图标路径

## 🎯 功能模块

### 模块1: 颜色解析 (`color_parser`)
- **输入**：HEX或RGB颜色字符串
- **输出**：颜色预览 + HEX/RGB/HSL格式
- **功能**：格式验证、颜色转换、复制到剪贴板

### 模块2: 图片色卡 (`image_colors`)
- **输入**：图片文件（JPG/PNG/GIF等）
- **输出**：图片主色调提取
- **功能**：图片上传、颜色分析、占比显示

### 模块3: UI色卡 (`ui_colors`)
- **数据**：预设的UI设计颜色
- **输出**：颜色卡片展示
- **功能**：分类展示、使用说明、快速复制

### 模块4: 传统色 (`traditional_colors`)
- **数据**：中国传统颜色库
- **输出**：传统颜色展示
- **功能**：朝代信息、拼音标注、文化背景

### 模块5: 渐变色 (`gradients`)
- **数据**：预设渐变色方案
- **输出**：渐变预览 + CSS代码
- **功能**：收藏管理、CSS复制、方向调整

### 模块6: 收藏功能 (`color_favorites`)
- **数据**：用户收藏的颜色和渐变色
- **输出**：收藏列表展示
- **功能**：添加/移除收藏、批量管理

## 🔄 数据流

```
用户输入
    ↓
功能路由 (根据输入类型选择功能)
    ↓
数据处理 (解析/提取/查询)
    ↓
UI渲染 (显示结果)
    ↓
用户交互 (复制/收藏/导出)
    ↓
状态更新 (更新收藏/历史)
```

## 🎨 设计系统

### 颜色变量
```css
:root {
  --bg-primary: #ffffff;      /* 主背景 */
  --bg-secondary: #f5f5f5;    /* 次要背景 */
  --text-primary: #1e1e1e;    /* 主要文字 */
  --accent-color: #0078d4;    /* 强调色 */
  /* ... 更多变量 */

```

### 组件规范
- **间距**：使用8px为基准单位
- **圆角**：4px/8px/12px三级圆角
- **阴影**：三级阴影系统
- **动画**：200ms过渡时间

- 侧边导航（可选）

### 移动端 (≤ 768px)
